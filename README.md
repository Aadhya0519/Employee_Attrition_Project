# Employee Attrition Prediction

## Description
This project predicts whether an employee is likely to leave the company using a Machine Learning model.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Project Structure
- data/ : Dataset
- models/ : Saved trained model
- notebooks/ : Sprint notebooks
- app.py : Streamlit application
